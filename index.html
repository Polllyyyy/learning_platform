О курсе — чему Вы научитесь

1. Введение

Узнаем, что такое HyperText Markup Language и что он определяет. Обсудим основные понятия Всемирной паутины: веб-страница, веб-сервер. Узнаем, где найти перечень всех HTML-элементов и научимся добавлять разметку в обычный текст. Узнаем, что такое атрибуты, и зададим их некоторым HTML-элементам на странице.

2. HTML5: База

Первое занятие, посвященное HTML, на котором мы познакомимся с языком гипертекстовой разметки - основой любого веб-сайта. Теги, атрибуты, особенности интерпретации, семантика и практика-практика-практика. Попутно освоишь основные инструменты веб-разработчика: редактор кода, отладчик.

3. Основы HTML5

В изучение основ языка HTML включены следующие шаги: элементы и их виды, списки, размещение изображений, адреса, ссылки и якоря, таблцицы, формы, создание гиперссылок и работа с ними.

4. Основы CSS3

В изучении основ языка CSS узнаем способы подключения, распространённые виды селекторов, включая селекторы псевдоклассов и псевдоэлементов (синтаксис CSS). Изучим оформление страниц: смена фона, текста, шрифта, единицы измерения, позиционирование и т.п.

5. Продвинутая вёрстка

В этой части будут рассмотрены CSS animation и @keyframes. Всё про CSS отступы: padding и margin. CSS transform - трансформация.

6. Погружение в адаптивность

Сайт, не адаптированный под мобильные устройства, просто не может считаться сайтом. Мы разберемся с темой media queries или медиа-запросов в CSS для создания отзывчивой (адаптивной) верстки. Ориентация на изучение Bootstrap в будущем
